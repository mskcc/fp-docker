How to Read a FACETS Plot

The FACETS (Fraction and Allele-Specific Copy number Estimates from Tumor/Normal Sequencing) plot is a powerful tool used to infer copy number alterations in tumor samples. This guide will walk you through the key sections of a FACETS plot and explain how to interpret them.

1. Copy Number Profile

The top section of the FACETS plot shows the inferred copy number profile of the tumor. This profile is a visual representation of the number of DNA copies present at each position in the genome.

  • Red line: The total copy number.
  • Blue line: The minor allele copy number (the number of copies of the less abundant allele).

2. LogR and BAF Tracks

The middle section of the plot contains the LogR and BAF tracks. These tracks represent the allelic imbalance and copy number variations between the tumor and normal samples.

  • LogR: Shows the log ratio of observed to expected coverage for each genomic position.
  • BAF (B-Allele Frequency): Displays the frequency of the minor allele at each position.

Example of a FACETS plot with labeled sections.

3. Interpreting Copy Number Variations

Copy number variations (CNVs) can be inferred from deviations in the red and blue lines in the copy number profile. Large shifts indicate potential gains or losses in the copy number, which may be important for understanding the tumor's genetic alterations.

4. Summary Table

The summary table at the bottom of the plot provides a numerical overview of the copy number alterations inferred from the plot. This table includes the estimated total and minor allele copy numbers across different genomic segments.

Conclusion

FACETS plots provide valuable insights into the genomic alterations in a tumor. By understanding the key sections of the plot and how to interpret copy number variations, you can better understand the genetic makeup of the tumor and its potential clinical implications.
